Via les fichiers vectoriels (IGN)

Introduction

L'exemple est bas sur les fichiers IGN disponibles gratuitement en ligne : ce sont les fichiers appels GEOFLA.

Il y a la ncessit des deux fusions pour rcuprer une data.frame avec les couleurs et les noms de dpartement.

Le paquet est rgdal. Sous Linux, il faut installer les paquets et pour pouvoir installer si une machine Linux.

Via les fichiers vectoriels (IGN)

Chargement

La lecture se fait comme ci-dessous en spcifiant le rpertoire avec les fichiers shapefiles.

dpts <- readOGR(
  dsn="Cartes IGN/France/DEPARTEMENT", 
  layer="DEPARTEMENT"
)
test <- match(dpts$CODE_DEPT,fusion$DEP)

Pour les couleurs on rcupre les positions (l'ordre) des dpartements avec la fonction match.

Impression
