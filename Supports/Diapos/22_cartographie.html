Via les fichiers vectoriels (IGN)

Introduction

L'exemple est basé sur les fichiers IGN disponibles gratuitement en ligne : ce sont les fichiers appelés GEOFLA.

Il y a la nécessité des deux fusions pour récupérer une data.frame avec les couleurs et les noms de département.

Le paquet est rgdal. Sous Linux, il faut installer les paquets et pour pouvoir installer si une machine Linux.

Via les fichiers vectoriels (IGN)

Chargement

La lecture se fait comme ci-dessous en spécifiant le répertoire avec les fichiers shapefiles.

dpts <- readOGR(
  dsn="Cartes IGN/France/DEPARTEMENT", 
  layer="DEPARTEMENT"
)
test <- match(dpts$CODE_DEPT,fusion$DEP)

Pour les couleurs on récupère les positions (l'ordre) des départements avec la fonction match.

Impression

par(mar=c(0,0,0,0))
plot(dpts,col=couleurs$couleur[test])
